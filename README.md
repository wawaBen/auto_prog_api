# auto_programming_api

# Getting Started with API with FASTAPI

This project was created from [How to Deploy a FastAPI](https://tutlinks.com/deploy-fastapi-app-on-google-cloud-platform/?expand_article=1).

# Content

This script works to decrypt and manipulate bin files for autoprogramming

# How to use

To launch it locally run 

### `uvicorn -b :8080 main:app`

then use postman to test it


# How to deploy

When ready you can deploy it with

### `gcloud app deploy`
